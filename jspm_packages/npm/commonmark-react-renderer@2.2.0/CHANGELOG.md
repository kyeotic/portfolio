# Change Log

All notable changes will be documented in this file.

## [2.1.0] - 2015-11-20

### Added

- Add support for specifying which types should be allowed - `allowTypes`/`disallowedTypes` (Espen Hovlandsdal)

## [2.0.2] - 2015-11-19

### Added

- Add support for hard linebreaks (marlonbaeten)

## [2.0.1] - 2015-10-22

### Changed
